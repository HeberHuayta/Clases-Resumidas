¿Quien es alt?

< video >

Sirve para un insertar video

Atributos

  • controls ( Muestra los controles del reproductor -play, pausa, volumen- )
  • autoplay ( El video se reproduce automáticamente )
  • muted ( El video empieza sin sonido )
  • loop ( Cuando termina, vuelve a empezar automáticamente )

Atencion! En muchos navegadores -loop- solo funciona si también está muteado, de lo contrario el video sera bloqueado
