Clase III

Layout

Lo siguiente es un ejemplo de diseño de pagina

< header >

Sirve para colocar el encabezado de la página o sección: logo, menú, título principal, navegación, etc.

< main >

Sirve para contener el contenido principal de la página, lo más importante de todo el sitio

< footer >

Sirve para poner el pie de página, información final como derechos, enlaces, contacto, redes, etc.

< nav >

Sirve para contener links para moverse por la página. Ej: menú principal, barra de búsqueda, botón de login, carrito, ajustes

< aside >

Sirve para mostrar contenido secundario o complementario, como barras laterales, anuncios, información extra

< article >

Sirve para contenido independiente y completo, como un post, noticia, comentario, tarjeta, producto, etc.

Ej: acá hay tres < article >
