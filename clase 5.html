Layout

Lo siguiente es un ejemplo de diseño de pagina


< header >

Sirve para colocar el encabezado de la página o sección: logo, menú, título principal, navegación, etc.


< main >

Sirve para contener el contenido principal de la página, lo más importante de todo el sitio


< footer >

Sirve para poner el pie de página, información final como derechos, enlaces, contacto, redes, etc.


< nav >

Sirve para contener links para moverse por la página. Ej: menú principal, barra de búsqueda, botón de login, carrito, ajustes


< aside >

Sirve para mostrar contenido secundario o complementario, como barras laterales, anuncios, información extra


< article >

Sirve para contenido independiente y completo, como un post, noticia, comentario, tarjeta, producto, etc.

Ej: acá hay tres < article >


< section >

Sirve para agrupar contenido relacionado dentro de una página, como capítulos, temas o partes de una misma sección

Ej: acá hay cuatro < section >


Lorem

Sirve para generar texto de relleno, para usarlo tenés que escribir "lorem" + (tab)

TIP: Podes personalizar la longitud del texto escribiendo la cantidad de palabras deseadas adelante de "lorem" → "lorem300"


-id- y -class-

-class- Sirve para agrupar elementos que queremos que compartan un mismo estilo o comportamiento, permitiendo seleccionarlos desde CSS o JavaScript

-id- Sirve para darle un identificador único a un elemento, este tambien puede ser seleccionado por < a > ademas de CSS y JS

Ej: haciendo click en esta imagen seras redirigido al < h1 > de la "Clase II"
